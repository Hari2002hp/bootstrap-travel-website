# 🌍 Travel Agency Landing Page

This project is a modern and fully responsive Travel Agency Landing Page developed using HTML5, CSS3, and Bootstrap 5. The website is designed with a clean, elegant, and user-friendly interface to provide an engaging travel booking experience.

The landing page includes a responsive navigation bar, an attractive hero banner, popular destinations, why choose us section, tour packages, customer testimonials, call-to-action, and a professional footer. Custom CSS animations, hover effects, glassmorphism, and responsive layouts are implemented to enhance the overall user experience.

The project is optimized for desktop, tablet, and mobile devices using Bootstrap's responsive grid system. It demonstrates modern front-end development practices and focuses on clean UI design without using JavaScript for interactive effects.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

## Features

- Responsive Design
- Modern Hero Section
- Popular Destinations
- Tour Packages
- Testimonials
- Call To Action
- Professional Footer
- Custom CSS Animations
- Mobile-Friendly Layout
